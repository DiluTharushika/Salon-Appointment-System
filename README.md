# Salon App

A full-stack salon management application currently under development.

## Project Status
🚧 Ongoing Project / Work in Progress

This project is being developed to manage salon-related operations such as appointments, services, customers, and administration features.

## Tech Stack
- **Frontend:** Next.js
- **Backend:** Python, FastAPI
- **Database:** PostgreSQL
- **Containerization:** Docker, Docker Compose

## Project Structure
- `frontend/` – frontend built with Next.js
- `backend/` – backend API built with Python and FastAPI
- `docker-compose.yml` – configuration for application services

## Planned Features
- Appointment booking
- Customer management
- Service management
- Authentication and authorization
- Admin dashboard

## Note
This project is still in active development, and some features may be incomplete or change over time.
